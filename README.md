 IoT Linked Collective Human Brain:

 Project Overview :
This project explores the concept of connecting human brain signals to the Internet using IoT technology. Brain signals are captured using EEG sensors, processed using a microcontroller, and transmitted over the Internet for analysis and control applications.

The system aims to create a collective intelligence platform where brain signals from multiple users can be collected, analyzed, and used to control devices or assist in decision-making systems.

 Objectives :
- To capture brain signals using EEG sensor
- To process brain signals using microcontroller
- To send brain data using IoT (Wi-Fi)
- To analyze brain signals using software
- To control devices using brain signals

 Components Used :
- EEG Sensor (NeuroSky / MindWave)
- ESP32 / NodeMCU
- Relay Module
- Wi-Fi Module
- Cloud Platform (ThingSpeak / Blynk)
- Jumper Wires
- Breadboard

 Technologies Used :
- Embedded C
- IoT (Wi-Fi / MQTT)
- Brain Computer Interface (BCI)
- Signal Processing
- Cloud Computing
- Python (for data analysis)

 Working Principle :
The EEG sensor reads brainwave signals such as attention and meditation levels. These signals are sent to a microcontroller like ESP32. The microcontroller sends the data to a cloud server using IoT. The data can be analyzed and used to control external devices like lights, wheelchair, or other automation systems.

 Applications :
- Brain Controlled Home Automation
- Assistive Technology for Disabled People
- Brain Signal Monitoring
- Smart Healthcare
- Human-Machine Interface
- Collective Intelligence Systems

  Future Scope :
